# Flask Session practica

Proyecto simple en Flask para demostrar el uso básico de **templates**, **formularios** y **manejo de sesiones**.

Este proyecto fue creado para practicar conceptos fundamentales de Flask Web antes de pasar a un enfoque API

---

## ¿Qué demuestra este proyecto?

- Creación de una aplicación Flask
- Uso de `render_template` con HTML
- Manejo de formularios (`request.form`)
- Uso de sesiones con `session`
- Flujo básico de autenticación:
  - login
  - perfil 
  - logout
- Redirecciones con `redirect` y `url_for`

---

## Funcionalidades

- Página de inicio
- Página de login con formulario
- Al iniciar sesión se guarda el usuario en sesión
- Página de perfil accesible solo si hay sesión activa
- Ruta de logout para cerrar sesión

**Nota:** No se implementa autenticación real ni seguridad avanzada.  
Es un proyecto de práctica.
